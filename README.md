# Sales & Profitability Performance Analysis

## Problem Statement
Businesses often experience revenue growth without proportional profit growth.
This project analyzes sales and profitability data to identify loss-making products,
regions, and discount strategies.

## Dataset
Sales transaction dataset containing order details, product categories,
regions, discounts, sales, and profit.

## Approach
- Data cleaning and preprocessing
- Revenue and profit trend analysis
- Regional and product-level profitability analysis
- Business interpretation of loss-making segments

## Tools Used
- SQL
- Python (pandas, numpy, matplotlib)
- Microsoft Excel
